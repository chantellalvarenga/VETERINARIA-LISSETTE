Nexpose

El producto de administración de vulnerabilidades Rapid7 Nexpose viene en varias ediciones con diferentes opciones de implementación:

Ultimate: se ofrece como producto de software, dispositivo virtual, dispositivo de hardware, nube privada o servicio administrado; proporciona todas las funciones con un número ilimitado de direcciones IP, usuarios y motores de análisis.
Enterprise: se ofrece como producto de software, dispositivo virtual, dispositivo de hardware, nube privada o servicio administrado; diseñado para organizaciones medianas y grandes, generalmente con equipos de seguridad; es escalable y admite un número ilimitado de IP, usuarios y motores de análisis.
Consultant: Se ofrece como producto de software o dispositivo virtual; diseñado para organizaciones que brindan consultoría en seguridad de TI; se instala en una sola computadora portátil, admite un motor de escaneo y escanea hasta 1024 IP.
Express: se ofrece como producto de software, dispositivo virtual o nube privada; diseñado para pequeñas organizaciones; admite un usuario, dos motores de escaneo y escanea hasta 1.024 IP.
Community: viene como un producto de software o un dispositivo virtual; un motor de escaneo para un solo usuario y escanea hasta 32 direcciones IP.

Posee una cierta limitante en cuanto a computadoras en las que se puede ocupar y ejecutar, pues puede ejecutarse tanto en sistemas operativos basados en Linux (Ubuntu Linux 18.04 LTS, Ubuntu Linux 16.04 LTS,Red Hat Enterprise Linux Server 6 - 8, CentOS 7, Oracle Linux 7, SUSE Linux Enterprise Server 12, openSUSE Leap 15) como en Microsoft Windows (Microsoft Windows Server 2019, Microsoft Windows Server 2016, Microsoft Windows Server 2012 R2, Microsoft Windows Server 2008 R2, Microsoft Windows 8.1), según documentación de la desarrolladora, los requerimientos mínimos del sistema donde se ejecutará son:
RAM:                 8 Gb
Espacio en Disco:         100 Gb
Procesador:             Dual-Core
Navegadores Compatibles:    Google Chrome, Mozilla Firefox
